livetexts — Self-hosted Liveblocks Alternative · npm package · Feb. 2026

[npm] · [github]

  • Open-source npm package for real-time collaborative rooms — presence, CRDT storage, collaborative text, and offline sync
  • One Durable Object per room on Cloudflare Workers — the same architecture Liveblocks runs in production
  • Yjs CRDT with bidirectional state-vector sync for lossless offline merge across tab closes and dropped connections
  • JWT auth signed locally — no round-trip to a central auth service. Zero idle cost via WebSocket hibernation
  • Stack: Cloudflare Workers, Durable Objects, Yjs, SQLite, TypeScript

Why this matters: Self-hostable means you own your infrastructure, your data, and your costs. The same real-time primitives Liveblocks sells — without the vendor lock-in.
