Fullstack Engineer

I own the entire stack. Frontend architecture, backend services, databases, CI/CD, deployment. I've shipped CRM platforms, real-time collaborative editors, chat systems over WebSockets, and marketing sites on Cloudflare Workers + KV.

Design Engineer

I sit at the intersection of design and code. I don't just implement designs — I think about how interfaces should feel. Drag-and-drop, real-time cursors, responsive layouts, loading states. The details that make software feel alive.

System Thinker

I think in systems, not features. Database schemas, CRDT conflict resolution, CI/CD pipelines, caching strategies. I read DDIA and OSTEP for fun and I believe the best UI is backed by the best architecture.
